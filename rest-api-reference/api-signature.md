# Signature

## PHP
`$securityKey` is generated by cpay.

```php
function genSign($param, $securityKey) {
  if (!is_array($param) || count($param) == 0) {
    return '';
  }

  ksort($param);
  $ps = '';
  foreach ($param as $k => $v) {
    if (!empty($v)) {
      $ps = $ps . "{$k}={$v}&";
    }
  }
  $ps = $ps.'key='.$securityKey;
  return hash_hmac("sha256", $ps, $securityKey);
}
```
